# Server Events

<Callout type="warning">
All these events are send by ox_core, you should not trigger them.
</Callout>

## ox:savedPlayers

When players are saved in database

```lua
function(playersSaved: number)
```

## ox:setActiveGroup

When the active group of a character has changed

```lua
function(playerId: number, groupName: string)
```

## ox:setGroup

When a character is added or removed from a group

```lua
function(playerId: number, groupName: string, grade?: number)
```

## ox:licenseAdded

When a character got a new license

```lua
function(playerId: number, licenseName: string)
```

## ox:licenseRemoved

When a character got one of his license removed

```lua
function(playerId: number, licenseName: string)
```

## ox:playerLogout

When a player logout, crash, disconnect etc

```lua
function(playerId: number, userId: number, charId: number)
```

## ox:createdCharacter

When a client created a new character

```lua
function(playerId: number, userId: number, charId: number)
```

## ox:playerLoaded

When a character is loaded

```lua
function(playerId: number, userId: number, charId: number)
```

## ox:deletedCharacter

When a character is deleted

```lua
function(playerId: number, userId: number, charId: number)
```

## ox:savedVehicles

When vehicles are saved in database

```lua
function(vehiclesSaved: number)
```

## ox:spawnedVehicle

When a `OxVehicle` has spawned

```lua
function(entityId: number, id: number)
```
