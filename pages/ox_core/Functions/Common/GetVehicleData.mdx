## Ox.GetVehicleData

Returns VehicleData for all vehicles, or optionally a specific model or array of models.

```lua
Ox.GetVehicleData(filter)
```

**Parameters**

- filter?: `string` | `string[]`

**Returns**

- `{ [string]: VehicleData }` | `VehicleData`

## VehicleData
- `object`
  - class: `VehicleClasses`
  - doors: `number`
  - make: `string`
  - name: `string`
  - price: `number`
  - seats: `number`
  - type: `string`
  - weapons?: `true`
