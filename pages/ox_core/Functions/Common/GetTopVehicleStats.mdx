## Ox.GetTopVehicleStats

Returns an object containing the top vehicle stats, either overall or for a specific category (land, air, sea).

```lua
Ox.GetTopVehicleStats(category)
```

**Parameters**

- category?: `string`

**Returns**

- `VehicleStats` | `{ [string]: VehicleStats }`

## VehicleStats
- `object`
  - acceleration: `number`
  - braking: `number`
  - handling: `number`
  - speed: `number`
  - traction: `number`
