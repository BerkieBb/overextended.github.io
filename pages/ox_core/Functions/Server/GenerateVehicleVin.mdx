## Ox.GenerateVehicleVin

Generate a unique vehicle identification number that doesn't exist in database.

```lua
Ox.GenerateVehicleVin(modelName)
```

**Parameters**

- modelName: `string`
  - Vehicle information (i.e. model and make) are used as part of the VIN.

**Returns**

- `string`
