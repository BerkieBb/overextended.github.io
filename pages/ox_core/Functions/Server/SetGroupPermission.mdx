## Ox.SetGroupPermission

Assigns a permission to a specified group and minimum grade, which will be inherited by higher grades.

```lua
Ox.SetGroupPermission(groupName, grade, permission, value)
```

**Parameters**

- groupName: `string`
- grade: `number`
- permission: `string`
- value: `allow` | `deny`
