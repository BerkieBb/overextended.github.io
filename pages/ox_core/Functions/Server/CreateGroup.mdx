## Ox.CreateGroup

Creates a new group from the given data, handling all database interactions internally.

```lua
Ox.CreateGroup(data)
```

**Parameters**

- data: `object`
  - name: `string`
  - label: `string`
  - grades: `object[]`
    - label: `string`
    - accountRole?: `string`
  - type?: `string`
  - colour?: `number`
  - hasAccount?: `boolean`
