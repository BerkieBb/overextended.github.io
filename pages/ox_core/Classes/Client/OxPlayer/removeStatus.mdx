# OxPlayer.removeStatus

<Callout>
  Documentation is work in progress as ox_core is still in development.<br />
  This is written for `ox_core v0.31.1`
</Callout>

A method used to remove from a specific status of the player.

```lua
player.removeStatus(statusName, value)
```

- statusName: `string`
- value: `number`

Return:

- success: `boolean`
