# OxPlayer.getGroupByType

<Callout>
  Documentation is work in progress as ox_core is still in development.<br />
  This is written for `ox_core v0.31.1`
</Callout>

A method used to retrieve a players groups with grade, filtered by type.

```lua
player.getGroupByType(type)
```

- type: `string`

Returns:

- groups: `table`
  - group: `string`
  - grade: `number`
