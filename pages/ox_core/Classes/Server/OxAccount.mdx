# OxAccount

- id: `number`
- balance: `number`
- isDefault: `number`
- label?: `boolean`
- owner?: `number`
- group?: `string`
- type: `string`, possible values: `personal`, `shared`, `group`
- role: `OxAccountRole`
- ownerName: `string`


**OxAccountRole**: `string`, possible values: `viewer`, `contributor`, `manager`, `owner`
